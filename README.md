# Peta-Interaktif-Sari-Natalia-Suku_Indonesia

Interactive web-based map showcasing the cultural diversity of Indonesian tribes across all 38 provinces. This project combines geographic visualization with educational content, allowing users to explore indigenous ethnic groups, traditional arts, languages, customs, and regional information through an intuitive interactive interface.

## Live Demo

https://sari-natalia-suku-indonesia.vercel.app/

---

## Overview

Indonesia is one of the most culturally diverse countries in the world, consisting of hundreds of ethnic groups spread across thousands of islands. This project was developed to provide an engaging and educational platform for learning about Indonesian tribes through an interactive map experience.

Users can select a province directly from the map or province list to view detailed information about its dominant ethnic group, cultural heritage, language, traditions, and other regional characteristics.

---

## Features

### Interactive Provincial Map

* Interactive map powered by Leaflet.js and OpenStreetMap
* Geographic visualization of all Indonesian provinces
* Clickable markers for quick exploration

### Province Explorer

* Browse all 38 Indonesian provinces
* Search provinces instantly using the built-in search feature
* Quick navigation between regions

### Tribe Information System

* Dominant ethnic group information for each province
* Historical and cultural descriptions
* Traditional arts and cultural heritage
* Local languages and religions
* Common occupations and livelihoods

### Detailed Information Panel

* Province overview
* Population data
* Area information
* Capital city information
* Cultural and ethnic descriptions
* Full-screen reading mode

### Modern User Experience

* Responsive design for desktop and mobile devices
* Dark mode and light mode support
* Animated loading screen
* Smooth transitions and modern interface components
